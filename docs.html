Troublemaker

Everything is not always ok

/info
Returns plain text diagnostic data including GOMAXPROCS, NumCPU, and Build Settings.
/logs
Returns the last log.size lines of application logs in plain text.
/slow
Streams a random emoji every 100ms.
ParameterTypeDefaultDescription
durationDuration5mHow long to keep the stream open (e.g., 10s, 2m).
/load/cpu
Starts the predefined cpu load pattern. Only a single load generator can run at one time.
/load/mem
Starts the predefined memory load pattern. Only a single load generator can run at one time.
/load/combined
Starts a combined cpu and memory load pattern. Only a single load generator can run at one time.
/nothing
Opens a TCP connection but sends nothing (no headers, no body).
ParameterTypeDefaultDescription
durationDuration5mTime to stall before hijacking and closing the connection.
/killhttp
Gracefully shuts down the HTTP server. The application will continue running but will no longer respond to HTTP requests.
/exit/[code]
Immediately terminates the troublemaker server process.
ParameterTypeDefaultDescription
codeInteger1The OS exit status code (0-127).
